📊 Customer Churn Prediction

Executive Summary

Customer churn directly impacts revenue and long-term profitability. This project builds a machine learning solution to predict churn and enable proactive retention strategies.

  • Dataset: ~12,000 customers
  • Best Model: Random Forest (tuned)
  • Threshold optimized to 0.3 for better recall
  • Key drivers: Contract, Tenure, Payment Behavior

The model enables businesses to identify at-risk customers early and take targeted action.

Introduction

Customer retention is critical for subscription-based industries. Acquiring new customers is significantly more expensive than retaining existing ones.

Machine learning enables:

  • Pattern detection in customer behavior
  • Predictive decision-making
  • Proactive retention strategies

This project focuses on leveraging data to reduce churn and improve long-term profitability.

Methods

Data Sources

  • 12,023 records
  • 21 features
  • Mixed categorical + numerical data

Data Processing

  • Removed customerID
  • Converted TotalCharges to numeric
  • One-hot encoding for categorical variables
  • Standard scaling for numerical features

Modeling Approach

  • Logistic Regression
  • Decision Tree
  • Random Forest
  • Gradient Boosting

Optimization

  • GridSearchCV for hyperparameter tuning
  • Threshold adjusted to 0.3
  • Recall prioritized over precision

Results

Descriptive Insights

  • ~37% customers churned → class imbalance
  • Low tenure customers churn more
  • Month-to-month contracts → highest churn
  • Electronic check users → high risk

Model Performance

Accuracy
~60–65%
Recall
High (priority)
ROC-AUC
Strong

Lower threshold improved recall → better churn detection at slight precision cost.

Feature Importance

  • Contract Type
  • Tenure
  • Monthly Charges

Visual Insights

  • Churn Distribution: ~37% churn → moderate risk
  • Contract: Month-to-month highest churn
  • Tenure: Early lifecycle = high churn
  • Payment: Electronic check = risky
  • ROC Curve: Strong predictive capability
  • Confusion Matrix: High recall, acceptable false positives

Conclusion

Machine learning can effectively predict churn and provide actionable insights.

  • Random Forest performs best overall
  • Early-stage customers are highest risk
  • Behavioral + financial factors drive churn

Recommendations

  • Promote long-term contracts
  • Encourage auto-payment adoption
  • Improve onboarding experience
  • Deploy real-time churn prediction
  • Run A/B testing on retention strategies
